# Партия 026: аватар 000042

## Scope

Ассет создан одним вызовом встроенного `imagegen` по production template из [avatar_generation_prompts.md](../avatar_generation_prompts.md) и продолжает [матрицу ассетов](../avatar_asset_matrix.md). До генерации read-only селектор подтвердил отсутствие WebP, записи runtime-каталога, batch note и временного исходника для этой строки. Применено правило баланса палитры Elf. В соответствии с текущим workflow визуальный QA и повторная генерация не выполнялись.

Исходный PNG создан в текущей сессии и сохранён встроенным генератором в локальном каталоге Codex вне репозитория. В Git добавляется только итоговый WebP.

## Техническая проверка

- исходный файл существует и успешно декодируется;
- исходник имеет формат PNG, размер `1254×1254` и непрозрачный режим RGB;
- итоговый файл имеет формат WebP, размер `512×512` и режим RGB;
- `AvatarId`, путь и metadata совпадают со строкой 42 матрицы;
- Elf Druid Female и остальные непокрытые сочетания продолжают использовать `avatar.system.unknown`.
