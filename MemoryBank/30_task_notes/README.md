# Task Notes

Этот раздел хранит локальные статусные заметки по крупным задачам и эпикам.

Vikunja остаётся источником истины по фактическому статусу карточек. Здесь фиксируются инженерный контекст, связанные документы и next steps, чтобы не восстанавливать картину по разрозненным обсуждениям.

## Формат

Каждая заметка должна содержать:

- цель;
- что готово;
- что не готово;
- связанные файлы;
- next steps.
