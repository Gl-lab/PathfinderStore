# Domain

Этот раздел хранит нормативные правила предметной области.

## Разделы

- [`character_creation/`](character_creation) — правила создания персонажа Pathfinder 2e, MVP-границы, целевая модель и каталоги Player Core / Core Rulebook.
- [`store/`](store) — целевая архитектура кампаний, партий, предметов, экземпляров, инвентаря, магазинов и торговли.

## Правило продвижения

Материалы из `../90_research/` становятся частью доменной нормы только после переноса или обобщения в этом разделе.
