# Pathfinder 2e Character Creation Catalog Inventory

Дата проверки: 2026-04-04.

Цель документа:

- быстро показать, какие каталоги уже есть в `MemoryBank` и кодовой базе;
- зафиксировать ответ на вопрос про `Skill list`.

Недостающие и частично готовые области теперь ведутся в [known_gaps.md](known_gaps.md).

## Короткий ответ

Общий `Skill list` уже оформлен отдельным каталогом: [`skill_catalog_player_core.md`](skill_catalog_player_core.md).

Этот файл оставлен как компактная карта готовности каталогов. Подробные gaps вынесены в [known_gaps.md](known_gaps.md).

## Что найдено частично или отсутствует

Смотреть [known_gaps.md](known_gaps.md).

## Что это значит для проекта

1. Для MVP это не критично, потому что в `domain_rules_mvp.md` skills, spells, equipment, deity и languages вынесены за рамки.
2. Для расширения Character Creation использовать [known_gaps.md](known_gaps.md) как актуальный список недостающих каталогов.

## Рекомендуемый следующий шаг

Если продолжать систематизацию, логично завести следующие отдельные документы:

1. `background_catalog_cross_source.md`
2. `spell_catalog.md`
3. `deity_catalog.md`
4. `equipment_starting_wealth_catalog.md`
5. `language_catalog.md`

Это даст единый формат для дальнейшего заполнения и снимет путаницу между “упоминается в правилах” и “уже документировано как каталог”.
